# BruteForceHTTP

BruteForceHTTP is a simple python script to bruteforce login credentials

## Usage

```bash
python3 bruteforcehttp.py [url] [<username>=$USER&<password>=$PASS&other..] [U=<USERNAME>] [P=<Password File>] [Error Message]
```
